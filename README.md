## Documentation

### Installation

```shell
$ git clone {repo url}
$ cd {reponame}
$ pip install -r requirements.txt
```

### How To Start

```shell
$ python main.py
```

### Dependency
- Python==3.10
- django==3.0.0
- requests^0.10.0
	- 0.10.0버전 이상이어야 함을 의미한다.
